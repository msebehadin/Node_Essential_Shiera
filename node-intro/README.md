# Node.js Introduction Module (node_intro)

This module is designed to provide a hands-on introduction to the Node.js runtime environment.

##  Objectives

1.  Understand the core concepts of Node.js (what it is, how it works, and its benefits).
2.  Execute a basic Node.js script (`index.js`).
3.  Write and run a simple HTTP server using the built-in `http` module (`server.js`).
4.  Summarize learning outcomes in `node_summary.md`.

##  File Structure Overview

* `index.js`: A basic script demonstrating Node.js execution.
* `server.js`: A simple HTTP server implementation.
* `node_summary.md`: A personal summary of key Node.js concepts and benefits.
* `.gitignore`: Specifies files/folders to ignore in Git (e.g., `node_modules`).
